## Project Overview

Algos is a collection of algorithm and data structure implementations in Python for learning and practice.

## Commands

```bash
# Run any algorithm file directly
python main.py
python CE1.py  # Code challenge exercises

# No dependencies required - pure Python
```

## Architecture

### Data Structures (`data_structures/`)
- **BST_Practice.py** - Binary Search Tree
- **SLL_practice.py** - Singly Linked List
- **DLL_practice.py** - Doubly Linked List
- **Stack_practice.py** - Stack implementation
- **Queue_practice.py** - Queue implementation
- **Heap_practice.py** - Heap (min/max)
- **HashMap_practice.py** - Hash Map
- **Graph_practice.py** - Graph traversal
- **Weighted_Graph.py** - Weighted graph algorithms

### Sorting Algorithms (`sort_algorithms/`)
- Bubble sort, Insertion sort, Selection sort, Merge sort

### Search Algorithms (`search_algorithms/`)
- Linear search, Binary search

### String Manipulation (`string_manipulation/`)
- String reversal, Vowel counting

### Code Challenges
- **CE*.py** - Various coding exercise solutions

## Tech Stack

- Python 3 (no external dependencies)
